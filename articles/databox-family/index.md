<h1>Azure Data Box</h1>
<p>Семейство Azure Data Box предлагает продукты с различной емкостью хранилища, которые позволяют передавать терабайты (ТБ) данных в Azure быстрым, недорогим и надежным способом. Корпорация Майкрософт ускоряет безопасную передачу данных, доставляя принадлежащие исключительно вам запоминающие устройства, что обеспечивает автономную передачу данных или передачу данных по сети.</p>
