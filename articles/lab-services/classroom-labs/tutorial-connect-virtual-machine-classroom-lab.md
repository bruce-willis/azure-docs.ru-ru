# <a name="tutorial-access-a-classroom-lab-in-azure-lab-services"></a>Руководство. Вход в лабораторию для аудитории в Службах лабораторий Azure
С помощью этого руководства вы (как учащийся) подключитесь к виртуальной машине (VM) в лаборатории для аудитории. 

Вот какие действия выполняются в этом руководстве:

> [!div class="checklist"]
> * использование ссылки для регистрации; 
> * Подключение к виртуальной машине

## <a name="use-the-registration-link"></a>Использование ссылки для регистрации

1. Перейдите по **URL-адресу регистрации**, который вам предоставил преподаватель или методист. 
2. Войдите в службу, используя учебную учетную запись, чтобы завершить процесс регистрации. 
3. После регистрации убедитесь, что отображается виртуальная машина для лаборатории, к которой у вас должен быть доступ. 
2. Подождите, пока виртуальная машина будет готова, а затем **запустите** ее.

    ![Запуск виртуальной машины](../media/tutorial-connect-vm-in-classroom-lab/start-vm.png)

## <a name="connect-to-the-virtual-machine"></a>Подключение к виртуальной машине

1. Выберите **Connect** (Подключиться) на плитке, которая представляет виртуальную машину в нужной лаборатории. 

    ![Подключение к виртуальной машине](../media/tutorial-connect-vm-in-classroom-lab/connect-vm.png)
2. Сохраните RDP-файл на жесткий диск и откройте его. 
3. Чтобы войти на виртуальную машину, введите **имя пользователя** и **пароль**, которые вам предоставил преподаватель или методист. 

## <a name="next-steps"></a>Дополнительная информация
В рамках этого руководства вы вошли в лабораторию для аудитории с помощью ссылки для регистрации, предоставленной преподавателем или методистом.
