# <a name="how-to-prepare-for-an-ssl-ip-address-change"></a>Подготовка к изменению IP-адреса SSL

Если вы получили уведомление о том, что IP-адрес SSL приложения Службы приложений Azure будет изменен, следуйте инструкциям в этой статье, чтобы освободить IP-адрес SSL и назначить новый.

## <a name="release-ssl-ip-addresses-and-assign-new-ones"></a>Освобождение IP-адресов SSL и назначение новых

1.  Откройте [портал Azure](https://portal.azure.com).

2.  В меню навигации слева выберите **Службы приложений**.

3.  Выберите в списке приложение Службы приложений Azure.

4.  Под заголовком **Параметры** щелкните **Параметры SSL** в области навигации слева.

5. В разделе "Привязки SSL" выберите запись имени узла. В открывшемся редакторе выберите **SSL на основе SNI** в раскрывающемся меню **Тип SSL** и щелкните **Добавить привязку**. Если появилось сообщение об успешном выполнении операции, это значит, что существующий IP-адрес освобожден.

6.  В разделе **Привязки SSL** снова выберите запись имени узла с сертификатом. В открывшемся редакторе выберите **SSL на основе IP-адреса** в раскрывающемся меню **Тип SSL** и щелкните **Добавить привязку**. Когда появится сообщение об успешном выполнении операции, будет получен новый IP-адрес.

7.  Если на портале регистрации домена (для стороннего поставщика DNS или Azure DNS) настроена запись A (запись DNS, указывающая непосредственно на ваш IP-адрес), измените существующий IP-адрес на созданный. Новый IP-адрес можно найти, следуя инструкциям в следующем разделе.

## <a name="find-the-new-ssl-ip-address-in-the-azure-portal"></a>Поиск нового IP-адреса SSL на портале Azure

1.  Подождите несколько минут, а затем откройте [портал Azure](https://portal.azure.com).

2.  В меню навигации слева выберите **Службы приложений**.

3.  Выберите в списке приложение Службы приложений Azure.

4.  Под заголовком **Параметры** в области навигации слева щелкните **Свойства** и найдите раздел **Виртуальный IP-адрес**.

5. Скопируйте IP-адрес и измените конфигурацию записи домена или механизм IP-адресации.
